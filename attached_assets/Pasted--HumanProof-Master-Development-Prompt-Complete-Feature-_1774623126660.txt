## FEATURE 3 — DISPLACEMENT TIMELINE FORECAST (NEW)

### What to build
After the user completes either calculator, show a line chart projecting their risk score over 1, 3, and 5 years. Based on a growth curve applied to their current score.

### Projection logic
```js
// Conservative growth model based on historical automation adoption curves
const projectRisk = (currentScore, years) => {
  // Scores above 70 grow faster (already in automation zone)
  // Scores below 30 grow slowly (human-critical roles)
  const accelerationFactor = currentScore > 70 ? 1.4 : currentScore > 40 ? 1.15 : 1.04;
  return Math.min(99, Math.round(currentScore * Math.pow(accelerationFactor, years / 3)));
};

// Output 3 scenarios: Optimistic, Base, Pessimistic
const scenarios = {
  optimistic: { factor: 0.85, label: "If you upskill actively" },
  base: { factor: 1.0, label: "Current trajectory" },
  pessimistic: { factor: 1.2, label: "If AI adoption accelerates" },
};
```

### UI
- Recharts LineChart with 3 lines (green=optimistic, gray=base, red=pessimistic)
- X axis: Now / 1 year / 3 years / 5 years
- Y axis: Risk score 0–100
- Annotation: "At current pace, your risk could reach [X] by [year]"
- CTA: "Start your upskilling plan now" (links to Feature 5)

### Component file: `DisplacementForecast.jsx`

## FEATURE 6 — SCORE DRIFT TRACKER (NEW — RETENTION)

### What to build
Store the user's score in localStorage with a timestamp. On return visits, compare to previous score and show change. Surface a "Your score changed" banner when drift > 5 points.

### Implementation
```js
// scoreStorage.js
export const saveScore = (score, source) => {
  const entry = {
    score,
    source, // 'job' | 'skill' | 'human-index'
    timestamp: new Date().toISOString(),
  };
  const history = JSON.parse(localStorage.getItem('hp_score_history') || '[]');
  history.push(entry);
  localStorage.setItem('hp_score_history', JSON.stringify(history));
};

export const getScoreHistory = () => {
  return JSON.parse(localStorage.getItem('hp_score_history') || '[]');
};

export const getScoreDrift = () => {
  const history = getScoreHistory();
  if (history.length < 2) return null;
  const latest = history[history.length - 1].score;
  const previous = history[history.length - 2].score;
  return {
    change: latest - previous,
    direction: latest > previous ? 'increased' : 'decreased',
    previousDate: history[history.length - 2].timestamp,
  };
};
```

### UI
- On app load: check for drift. If drift > 5, show a banner: "Your risk score increased by 8 points since your last visit — here's why"
- History view: small line chart of all past scores over time
- Explanation of why scores drift (link to AI news feed)
